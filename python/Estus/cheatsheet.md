# Markdown Cheatsheet
 Quick guide for me to reference markdown syntax
 ## Element	Markdown Syntax
 ### Headings:
```
  # H1
  ## H2
  ### H3
``` 

# Non-Header:
* Bold:	**bold text**
* Italic:	*italicized text*
* Blockquote:	
> blockquote
* Ordered List	
> 1. First item
> 2. Second item
> 3. Third item
* Unordered List	
> - First item
> - Second item
> - Third item
> * slightly separated
> * from other items

* Code:	`code`
* Horizontal Rule	:
---
* Link	[title](https://www.example.com)
* Image	![alt text](image.jpg)
